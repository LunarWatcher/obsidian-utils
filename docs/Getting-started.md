# Getting started

## Installation

### Manually

From [Releases](https://github.com/LunarWatcher/obsidian-utils/releases) on GitHub, download the bundled .zip file, unzip it and drop it into your vault (`.obsidian/plugins/livi-utils`). You can also download the individual files manually; it's all the same files, but the `.zip` is shipped for more convenient manual use.

If you prefer, you can alternatively [download a nightly version](https://github.com/LunarWatcher/obsidian-utils/actions/workflows/build.yml), though these aren't as stable as versioned releases.

### Via BRAT 

BRAT might also work, but has not been tested. If it doesn't work, consider opening an issue.

### Obsidian plugin registry

[https://community.obsidian.md/plugins/livi-utils](https://community.obsidian.md/plugins/livi-utils)
